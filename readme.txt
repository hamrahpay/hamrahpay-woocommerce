افزونه درگاه پرداخت آنلاین همراه پی

== Description ==
با استفاده از این افزونه میتوانید به راحتی فروشگاه ووکامرس خود را به درگاه پرداخت آنلاین همراه پی متصل نمایید.

== Installation ==
1. پوشه \'woo-hamrahpay-payment-gateway\' را در مسیر \'/wp-content/plugins/\' آپلود کنید
2. افزونه را از طریق منوی \'افزونه ها\' در وردپرس فعال کنید
3. تنظیمات افزونه را می توانید از طریق قسمت تنظیمات ووکامرس / پیکربندی/ درگاه های پرداخت انجام دهید
